两个玩家将一摞nnn个石子堆上进行游戏，玩家一轮可以从石堆中移除 aaa 个石子或 bbb 个石子，不能行动的玩家输。

寻找石子堆的石子数量 nnn，使得无论第一个玩家如何行动，第二个玩家总能获胜

*800

只要一共有 a+ba+ba+b 个石子，那么先手必输

CF1842A. Tenzing and Tsondu

Tsondu 和 Tenzing 正在玩纸牌游戏。Tsondu 有 nnn 个能量为 a1,a2,…,ana_1, a_2, \ldots,a_na1​,a2​,…,an​ 怪物，Tenzing 有 mmm 个能量为 b1,b2,…,bmb_1, b_2, \ldots, b_mb1​,b2​,…,bm​的怪物

Tsondu 和 Tenzing 轮流行动，Tsondu 先手。当能量为 xxx 的怪物选择攻击能量为 yyy 的怪物时，两个怪物的能量变为 x−yx-yx−y和y−xy-xy−x，当能量小于 0 时怪物死亡

怪物全部死亡的一方输。问 Tsondu 和 Tenzing 在最优决策下谁获胜（或平局）

*800

考虑攻击时存在怪物死亡时会出现伤害溢出的情况，x和y的变化等价于 x−min⁡(x,y)x - \min(x,y)x−min(x,y)， y−min(x,y)y - min(x,y)y−min(x,y)

所以只需要比较 ∑a\sum a∑a 和 ∑b\sum b∑b 即可

Alice 先手，双方轮流行动，不能行动的人输。假设两人行动最优，输出最终谁获胜

CF1834C. Game with Revesing

Alice 和 Bob 正在玩游戏，他们共有两个长度为 nnn 的由小写字母组成的字符串 SSS 和 TTT。Alice 和 Bob 轮流行动。

  • Alice 每回合选择 i∈[1,n]i \in [1,n]i∈[1,n] 和一个字符串 SSS 或 TTT 和一个小写字母 ccc，并将选择的字符串中第 iii 个符号替换成 ccc
  • Bob 选择一个字符串 SSS 或 TTT，并将其反转

游戏将进行到 S=TS = TS=T为止。

定义游戏的持续时间为 Alice 和 Bob 行动的总次数。Alice 想最小化游戏时间，但 Bob 要最大化时间。当双方都最优行动时，游戏将在多少回合后结束

*1200

对于 Bob而言，他的反转操作几乎无用，无论反转谁反转两次操作都等于没有反转，而 Alice 可以将 c 变为 c 来跳过1回合来让 Bob 使字符串反转

分别统计在开始反转和没反转下 Alice 需要操作的次数（即 Alice 第一回合是否跳过），最后算出操作次数取 min 即可。

如果 Alice 第一回合没有跳过，那么游戏一共进行了 cnt+cnt−(cntmod  2)cnt + cnt - (cnt \mod 2)cnt+cnt−(cntmod2) 次

如果跳过了第一回合，那么一共进行了 cnt+cnt−(1−cntmod  2)cnt + cnt - (1 - cnt \mod 2)cnt+cnt−(1−cntmod2)（加上跳过一回合的操作）
